## üé® Design System

- **Light Mode**: Clean white backgrounds with black accents
- **Dark Mode**: Deep blacks with subtle grays
- **Accent**: Dynamic based on theme
- **Border**: Subtle separators with low opacity

### Typography
- **Font**: Inter (Google Fonts)
- **Weights**: 300, 400, 500, 600, 700, 800
- **Scale**: Responsive font sizing
- **Line Height**: 1.6 for body, 1.2 for headings

### Spacing
- **Base Unit**: 1rem (16px)
- **Scale**: 0.5rem, 1rem, 1.5rem, 2rem, 3rem
- **Container**: 1200px max-width
- **Gaps**: 1.5rem for grids, 1rem for flex

### Shadows
- **Small**: 0 4px 16px rgba(0, 0, 0, 0.08)
- **Medium**: 0 8px 20px rgba(0, 0, 0, 0.15)
- **Large**: 0 20px 40px rgba(0, 0, 0, 0.16)
- **Hover**: Increased shadow on interaction

### Border Radius
- **Small**: 6px-8px (inputs, badges)
- **Medium**: 10px-12px (cards, buttons)
- **Large**: 16px (modals, major containers)
- **Pill**: 50px (CTA buttons)

### Transitions
- **Fast**: 0.3s ease
- **Medium**: 0.4s cubic-bezier
- **Slow**: 0.6s ease
- **Spring**: cubic-bezier(0.34, 1.56, 0.64, 1)
